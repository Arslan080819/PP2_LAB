# Practice 10 - Pygame Projects

## 1. Racer Game

A car racing game where you avoid enemy cars and collect coins.

### Features
- Control a green car using arrow keys or WASD
- Avoid red enemy cars
- Collect yellow coins for points
- Score display in top right corner
- Coin counter in top right corner
- Increasing difficulty with more enemies
- Game over and restart functionality


- Use **Arrow Keys** or **WASD** to move left/right
- Avoid enemy cars
- Collect coins ($)
- Press **R** to restart after game over
- Press **ESC** to quit

## 2. Snake Game

The classic Snake game with added features like levels and wall collision.

### Features
- Classic snake gameplay
- Wall collision detection (game over if hit wall)
- Food generation that doesn't spawn on snake or walls
- Level system - level up every 4 foods eaten
- Speed increases with each level
- Score and level counters displayed on UI bar
- Game over and restart functionality



## 3. Paint Application

A simple drawing application with multiple tools and color selection.

### Features
- **Brush Tool**: Draw freehand with selected color
- **Rectangle Tool**: Click and drag to draw rectangles
- **Circle Tool**: Click and drag to draw circles
- **Eraser Tool**: Erase parts of your drawing
- **Clear Button**: Clear entire canvas
- Color palette with 10 colors
- Selected tool and color highlighting

### How to Use
```bash
cd paint
python paint.py
```
- Select a tool from the bottom toolbar
- Select a color from the color palette
- Click and drag on canvas to draw
- For rectangles/circles: drag to set size
- Press **ESC** to quit
