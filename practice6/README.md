# Practice 6 – Python File Handling and Built-in Functions

This repository contains examples of:

• File handling (read, write, append, copy, delete)  
• Directory management using os and shutil  
• Built-in Python functions such as map, filter, reduce, enumerate, zip, sorted  

Structure:

file_handling/ – file operations  
directory_management/ – working with folders  
builtin_functions/ – examples of common Python built-in functions
