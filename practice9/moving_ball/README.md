# Moving Ball Game

A simple Pygame game with a red ball that moves on arrow key presses.


## Controls

| Key | Action |
|-----|--------|
| **↑** | Move up 20 px |
| **↓** | Move down 20 px |
| **←** | Move left 20 px |
| **→** | Move right 20 px |
| **Q** | Quit |

## Features
- Red ball (50×50 px, radius 25) on white background
- Moves exactly 20 pixels per key press
- Ball cannot leave the screen (off-screen moves are silently ignored)
- Live position display (x, y coordinates)
- Subtle grid background for visual reference
- Shadow + highlight for polished look
